Useful Commands

  1. Delete all tables

    python manage.py syncdb Creates the database tables for all apps in INSTALLED_APPS whose tables have not already been created.

  2. delete all data in all tables

    python manage.py flush returns the database to the state it was in immediately after syncdb was executed

  3. Create all tables as defined in the model

    python manage.py sqlclear will print the sql statement to drop all tables

  4. Totally Reset DB(Be careful, This is the panacea of db problem at the cost of losing all data)

    • Delete Database file
    • python manage.py migrate to recreate the database tables
